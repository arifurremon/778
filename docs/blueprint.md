# **App Name**: The Chattala

## Core Features:

- Splash Animation: Gracefully animate 'The Chattala' brand name upon initial page load, utilizing Framer Motion.
- Placeholder SVG Logo: Use a placeholder for an SVG logo during the splash animation.
- New User Registration: Provide a clean, email/password based sign-up form for new users.
- User Login: Enable returning users to log in securely with email and password.
- Session Management: Implement robust logic for maintaining and validating user sessions, ensuring persistence for authenticated users.
- Authenticated Redirect: Automatically redirect logged-in users directly to their dashboard, bypassing the login page.
- Mobile Responsiveness: Ensure the entire entry flow, including splash screen and authentication forms, is fully responsive across all device sizes.

## Style Guidelines:

- Default Theme: Dark Mode by default, utilizing a sophisticated palette of deep charcoal backgrounds (#121212) and crisp, light grey text (#E0E0E0) for a premium feel.
- Primary Accent Color: Sophisticated deep indigo (#5A637D) for primary interactive elements and branding, providing contrast against the dark background.
- Secondary Accent Color: Subtle cyan (#61B3CC), used sparingly for highlights and interactive elements to create visual interest.
- Typography Font: All text uses 'Inter', a modern sans-serif typeface, loaded correctly from Google Fonts, providing a clean, objective, and professional look suitable for both headlines and body text.
- Use a set of minimal, modern line icons for navigation and interactive elements, maintaining a professional and uncluttered appearance.
- Employ a sparse, grid-based layout with ample white space, ensuring content breathability and an organized, high-end feel, similar to Linear or Vercel.
- Animations: All transitions and UI changes, particularly the splash animation and form interactions, will be smooth, subtle, and fluid, leveraging Framer Motion for a buttery-smooth user experience.
